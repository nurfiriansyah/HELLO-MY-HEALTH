# HELLO-MY-HEALTH
 SDGs adalah program global untuk pembangunan berkelanjutan hingga 2030. Tugas besar ini fokus pada poin 7, yaitu energi bersih dan terjangkau. Kelompok membuat program C++ “Sistem Simulasi Manajemen Energi” yang mensimulasikan produksi, penggunaan, biaya, efisiensi, serta prediksi kebutuhan energi suatu wilayah.
